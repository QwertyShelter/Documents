# 项目太大了，想 "按照关键词搜索文本"

```cmd
git grep -n "solvePnP\|solvePnPRansac\|PnP\|RANSAC\|CameraLoss\|find_opt_scaling\|weiszfeld"
```
这样可以搜索 git 的提交记录，会为你检索关键词所在的文件。 `-n` 表示显示行号

```cmd
grep -RIn --line-number -e "solvePnP" -e "solvePnPRansac" -e "PnP" -e "RANSAC" -e "CameraLoss" -e "find_opt_scaling" -e "weiszfeld" .
```
作用：递归搜索当前目录及子目录中的所有文件。参数含义：

-R：递归搜索

-I：忽略二进制文件

-n：显示行号

-e：指定多个搜索模式

```cmd
Select-String -Path .\**\*.py -Pattern "solvePnP" -List
Select-String -Path .\**\*.py -Pattern "CameraLoss" -List  
Select-String -Path .\**\*.py -Pattern "find_opt_scaling" -List
```
作用：在Python文件中搜索特定模式
